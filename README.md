# Project setup

To run the project please follow the steps below:

1. Clone the repo

```console
git clone https://github.com/angelojea/next-nest-template.git
```

2. Go to the root folder and install dependencies:

```console
npm run setup
```

3. Start both client and server:

```console
npm start
```
